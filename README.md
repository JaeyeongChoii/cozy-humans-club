## 개요

코지 휴먼스 클럽(Cozy Humans Club)은 React Native 기반 SNS 모바일 애플리케이션으로,
사용자가 비교적 안전한 환경에서 자신의 생각을 공유하고 상호작용할 수 있도록 설계된 서비스입니다.
본 프로젝트는 팀 단위로 진행된 공동 개발 프로젝트이며, 저는 프론트엔드 파트를 담당했습니다.
본 레포지토리는 개인 포트폴리오 정리 목적으로 관리되고 있습니다.

---

## 주요 기능

### 1. 인증 및 접근 제어
- Google / Apple / Discord 로그인을 OAuth 2.0 기반으로 구현
- 클라이언트에 API Key나 Client Secret을 포함하지 않기 위해 모든 인증 로직을 백엔드 서버 위임 방식으로 설계
- 가입 인원 제한을 위한 정원 관리 로직 구현(Splash / OverLimit 화면을 통해 현재 상태를 사용자에게 명확히 노출)

### 2. 게시글 및 미디어 처리
- 이미지·동영상 업로드 및 미리보기 기능 구현
- 외부 링크에 대한 Link Preview UI 제공
- 간단한 의견 수렴을 위한 투표 기능 UI 구현

### 3. 게시글 상호작용
- 댓글 작성 및 게시글 인용 기능
- 커뮤니티 운영을 고려한 신고 기능 구현

## 팀 개발 및 협업 방식
- 구성: 프론트엔드 2인, 백엔드 2인, 기획·디자인 2인
- 하나의 화면/파일을 여러 명이 함께 수정하는 상황이 잦았기 때문에 역할을 “파일 소유”가 아닌 기능 책임 단위로 나누어 협업


---


##  개발 과정에서의 고민과 배운 점

### 클라이언트 보안 강화
- 앱 내에 API Key나 DB 접속 정보를 하드코딩하지 않기 위해 고민했습니다.
- 결과적으로 클라이언트는 오직 백엔드 서버(BASE_URL)와만 통신하고, 모든 민감한 인증 절차는 서버 위임 방식으로 구현하였습니다.

### 크로스 플랫폼 일관성 유지
- Android와 iOS의 UI 렌더링 차이(특히 KeyboardAvoidingView 등)를 해결하기 위해 Platform.OS 분기 처리를 적극 활용했습니다.
- 특히 스플래시 화면과 로그인 버튼 배치에서 기기별 최적화된 UX를 제공하기 위해 노력했습니다.

### 사용자 경험 (UX) 디테일
- 제스처 충돌 해결: ScrollView와 BottomSheet 간의 제스처 경합 문제를 해결하여 부드러운 스크롤 경험을 구현했습니다.
- 감성적 UI: galmuri 폰트를 적용하고 컬러 가이드를 준수하며 앱만의 독특한 감성과 색감을 일관되게 전달하고자 노력했습니다.
